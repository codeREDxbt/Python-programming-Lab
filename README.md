# Python Programming Lab

This repository contains lab exercises for a Python programming course.

## Structure
- `Lab 1/Calorietracker.py`: A calorie tracker script.
- `Lab 2/Gradebook.py`: A gradebook management script.

## Getting Started
1. Ensure you have Python 3.10+ installed.
2. Run the scripts from their respective folders:
   - `python "Lab 1/Calorietracker.py"`
   - `python "Lab 2/Gradebook.py"`

## Notes
- Scripts are self-contained and require no external dependencies.
- Feel free to submit issues or pull requests for improvements.
